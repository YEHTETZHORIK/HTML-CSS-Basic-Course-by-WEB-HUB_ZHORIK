Colors

HTML နှင့် CSS တွင် အရောင်များကို သတ်မှတ်ရန် အများဆုံး အသုံးပြုသော နည်းလမ်းသုံးခုရှိသည် - အမည်အရောင်များ (color names), Hexadecimal (hex) အရောင်များ, နှင့် RGB အရောင်များဖြစ်သည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Colors Example</title>
    <style>
        .red { color: red; }
        .green { color: green; }
        .blue { color: blue; }
    </style>
</head>
<body>
    <p class="red">This is a red text.</p>
    <p class="green">This is a green text.</p>
    <p class="blue">This is a blue text.</p>
</body>
</html>

Result

This is a red text.

This is a green text.

This is a blue text.

ရှင်းပြချက်

HTML နှင့် CSS တွင် အမည်အရောင်များကို အသုံးပြု၍ အရောင်များကို သတ်မှတ်နိုင်သည်။ နမူနာ - red, green, blue စသည်ဖြစ်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <p> tag များကို အသုံးပြုပြီး စိတ်ကြိုက် အရောင်များဖြင့် စာသားများကို ရေးသားပါ။
  2. စိတ်ကြိုက် အရောင်များကို သတ်မှတ်ရန် CSS ကို အသုံးပြုပြီး အရောင်များကို သတ်မှတ်ပါ။
  3. အရောင်အသစ်များကို ရေးသားပြီး နမူနာအဖြစ် ဖော်ပြပါ။
