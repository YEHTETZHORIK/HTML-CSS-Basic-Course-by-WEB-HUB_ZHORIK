Fonts

HTML နှင့် CSS တွင် စာလုံးပုံစံများကို သတ်မှတ်ရန် font-family property ကို အသုံးပြုသည်။ အဓိကအသုံးများသော စာလုံးပုံစံများမှာ Arial, Verdana, Tahoma, Georgia, Times New Roman စသည်ဖြစ်သည်။ Google Fonts ကို အသုံးပြု၍ ပိုမိုစိတ်ကြိုက် စာလုံးပုံစံများကို သတ်မှတ်နိုင်သည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Fonts Example</title>
    <style>
        .font-roboto {
            font-family: 'Roboto', sans-serif;
        }
        .font-oswald {
            font-family: 'Oswald', sans-serif;
        }
    </style>
</head>
<body>
    <p class="font-roboto">This is Roboto font.</p>
    <p class="font-oswald">This is Oswald font.</p>
</body>
</html>

Result

This is Roboto font.

This is Oswald font.

ရှင်းပြချက်

HTML နှင့် CSS တွင် font-family property ကို အသုံးပြုပြီး စာလုံးပုံစံများကို သတ်မှတ်နိုင်သည်။ Google Fonts ကို အသုံးပြု၍ ပိုမိုစိတ်ကြိုက် စာလုံးပုံစံများကို ရယူနိုင်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး ပုံမှန် စာလုံးပုံစံတစ်ခုနှင့် Google Fonts တစ်ခုကို သုံးပြီး စာသားများကို ဖော်ပြပါ။
  2. Google Fonts တွင် ရှာဖွေပြီး စိတ်ကြိုက် စာလုံးပုံစံကို ရွေးချယ်ပါ။
  3. CSS သုံး၍ စာသားများကို စိတ်ကြိုက် စာလုံးပုံစံများဖြင့် ဖော်ပြပါ။
