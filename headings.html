Heading Tags (h1 - h6)

HTML တွင် Heading Tags (h1 မှ h6) သည် စာမျက်နှာ၏ ခေါင်းစဉ်များကို ဖော်ပြရန် အသုံးပြုသည်။ h1 သည် အကြီးဆုံး အရေးကြီးဆုံးသော ခေါင်းစဉ်ကို ပြသပြီး h6 သည် အငယ်ဆုံး ခေါင်းစဉ်ကို ပြသည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Heading Tags Example</title>
</head>
<body>
    <h1>This is an h1 heading</h1>
    <h2>This is an h2 heading</h2>
    <h3>This is an h3 heading</h3>
    <h4>This is an h4 heading</h4>
    <h5>This is an h5 heading</h5>
    <h6>This is an h6 heading</h6>
</body>
</html>

This is an h5 heading
This is an h6 heading

ရှင်းပြချက်

HTML တွင် h1 မှ h6 ထိ Heading Tags များကို အသုံးပြုပြီး စာမျက်နှာ၏ ခေါင်းစဉ်များကို ပြသနိုင်သည်။ h1 သည် အရေးကြီးဆုံး အကြီးဆုံး ခေါင်းစဉ်ကိုပြသည်။ h6 သည် အငယ်ဆုံး ခေါင်းစဉ်ကို ပြသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <h1> မှ <h6> ထိ Heading Tags များကို အသုံးပြုပြီး သင်၏ အကြိုက်ဆုံး အစားအစာများကို စာရင်းပြုစုပါ။
  2. Heading Tags များအား လိုက်လျောညီထွေဖြစ်အောင် အရောင်အသုံးပြု၍ ဖော်ပြပါ။
  3. Heading Tags (h1 မှ h6) အား အသီးသီးအသုံးပြုပြီး သင်၏ မိသားစုဝင်များ၏ အမည်များကို စာရင်းပြုစုပါ။
