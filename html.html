What is .HTML

.html ဆိုသည်မှာ HyperText Markup Language အတွက် ဖိုင်အတိုးဖြစ်ပြီး ဝဘ်ဘရောက်ဆာများတွင် ဖတ်ရှုနိုင်သော ဖိုင်များကို ဖန်တီးရန် အသုံးပြုသည်။ .html ဖိုင်များသည် အချက်အလက်များကို စနစ်တကျ ဖော်ပြနိုင်စေရန် HTML ကို အသုံးပြုသည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Sample Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Result

Hello, World!

This is a paragraph.

ရှင်းပြချက်

.html ဖိုင်ကို ဖန်တီးရန်အတွက်, မည်သည့် text editor ကိုမဆို သုံးနိုင်ပြီး ဖိုင်ကို .html အတိုးဖြင့် သိမ်းဆည်းနိုင်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <h1> နှင့် <p> tag များကို အသုံးပြုပြီး မိမိကိုယ်ကို မိတ်ဆက်ပါ။
  2. ဖိုင်ကို .html ဖိုင်အတိုးဖြင့် သိမ်းဆည်းပြီး ဝဘ်ဘရောက်ဆာတွင် ဖွင့်ကြည့်ပါ။
