Images

HTML တွင် ပုံများကို <img> tag ကို အသုံးပြုပြီး ဖော်ပြသည်။ <img> tag ၏ src attribute သည် ပုံဖိုင်၏ လိပ်စာကို ဖော်ပြရန် အသုံးပြုသည်။ alt attribute သည် ပုံကို ဖော်ပြ၍မရပါက ဖော်ပြရန် စာသားကို ရေးသားရန် အသုံးပြုသည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Images Example</title>
</head>
<body>
    <img src="https://www.example.com/image.jpg" alt="Example Image">
</body>
</html>

ရှင်းပြချက်

HTML တွင် <img> tag ကို အသုံးပြုပြီး ပုံများကို ဖော်ပြနိုင်သည်။ <img> tag ၏ src attribute ကို အသုံးပြုပြီး ပုံ၏ URL ကို သတ်မှတ်ပြီး alt attribute ကို အသုံးပြုပြီး ပုံကို မဖော်ပြနိုင်ပါက ဖော်ပြမည့် စာသားကို ရေးသားနိုင်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <img> tag ကို အသုံးပြုပြီး သင်၏ အကြိုက်ဆုံး ပုံတစ်ပုံကို ဖော်ပြပါ။
  2. src attribute ကို အသုံးပြုပြီး ပုံ၏ URL ကို သတ်မှတ်ပါ။
  3. alt attribute ကို အသုံးပြုပြီး ပုံကို မဖော်ပြနိုင်ပါက ဖော်ပြမည့် စာသားကို ရေးသားပါ။
