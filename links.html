Links

HTML တွင် လင့်ခ်များကို <a> tag ကို အသုံးပြုပြီး ရေးသားသည်။ <a> tag သည် "anchor" အတွက် ရည်ရွယ်ပြီး လင့်ခ်တစ်ခုကို သတ်မှတ်ရန် အသုံးပြုသည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Links Example</title>
</head>
<body>
    <a href="https://www.example.com">Visit Example.com</a>
</body>
</html>

Result

Visit Example.com

ရှင်းပြချက်

HTML တွင် <a> tag ကို အသုံးပြုပြီး လင့်ခ်များကို ရေးသားနိုင်သည်။ <a> tag ၏ href attribute ကို အသုံးပြုပြီး လင့်ခ်ရဲ့ URL ကို ဖော်ပြနိုင်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <a> tag ကို အသုံးပြုပြီး သင်၏ အကြိုက်ဆုံး ဝဘ်ဆိုဒ်များကို လင့်ခ်ပြုလုပ်ပါ။
  2. ထို HTML ဖိုင်ကို .html ဖိုင်အတိုးဖြင့် သိမ်းဆည်းပါ။
  3. ဖန်တီးထားသော HTML ဖိုင်ကို ဝဘ်ဘရောက်ဆာတွင် ဖွင့်ကြည့်ပါ။
