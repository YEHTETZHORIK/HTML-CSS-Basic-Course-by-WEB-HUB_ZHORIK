Lists

HTML တွင် စာရင်းများကို <ul> (unordered list) နှင့် <ol> (ordered list) tag များကို အသုံးပြုပြီး ရေးသားသည်။ <li> tag သည် "list item" အတွက် ရည်ရွယ်ပြီး စာရင်းအချက်တစ်ခုစီကို သတ်မှတ်ရန် အသုံးပြုသည်။

Unordered List Example

<!DOCTYPE html>
<html>
<head>
    <title>Unordered List Example</title>
</head>
<body>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>
</body>
</html>

Ordered List Example

<!DOCTYPE html>
<html>
<head>
    <title>Ordered List Example</title>
</head>
<body>
    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>
</body>
</html>

Result

Unordered List

  • Item 1
  • Item 2
  • Item 3

Ordered List

  1. First item
  2. Second item
  3. Third item

ရှင်းပြချက်

HTML တွင် <ul> tag ကို အသုံးပြုပြီး အစီအစဉ်မရှိသော စာရင်းများကို ဖော်ပြနိုင်သည်။ <ol> tag ကို အသုံးပြုပြီး အစီအစဉ်ရှိသော စာရင်းများကို ဖော်ပြနိုင်သည်။ <li> tag ကို အသုံးပြုပြီး စာရင်းအချက်တစ်ခုစီကို သတ်မှတ်နိုင်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <ul> tag ကို အသုံးပြုပြီး သင်၏ အကြိုက်ဆုံး အစားအစာများကို စာရင်းပြုစုပါ။
  2. <ol> tag ကို အသုံးပြုပြီး သင်၏ နေ့စဉ်လုပ်ဆောင်မှုများကို စာရင်းပြုစုပါ။
  3. စိတ်ကြိုက် စာရင်းတစ်ခုကို ဖန်တီးပြီး <li> tag များကို အသုံးပြုပြီး အချက်တစ်ခုစီကို ရေးသားပါ။
