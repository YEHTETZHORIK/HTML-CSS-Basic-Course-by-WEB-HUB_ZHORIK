Paragraphs

HTML တွင် စာပိုဒ်များကို <p> tag ကို အသုံးပြုပြီး ရေးသားသည်။ <p> tag သည် "paragraph" အတွက် ရည်ရွယ်ပြီး စာပိုဒ်တစ်ခုစီကို သတ်မှတ်ရန် အသုံးပြုသည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Paragraphs Example</title>
</head>
<body>
    <p>This is the first paragraph.</p>
    <p>This is the second paragraph.</p>
    <p>This is the third paragraph.</p>
</body>
</html>

Result

This is the first paragraph.

This is the second paragraph.

This is the third paragraph.

ရှင်းပြချက်

HTML တွင် <p> tag ကို အသုံးပြုပြီး စာပိုဒ်များကို ရေးသားနိုင်သည်။ စာပိုဒ်များကို ခွဲခြားဖော်ပြရန် <p> tag ကို သုံးပါ။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <p> tag များကို အသုံးပြုပြီး သင်၏ နေ့စဉ်လုပ်ဆောင်မှုများကို စာရင်းပြုစုပါ။
  2. ထို HTML ဖိုင်ကို .html ဖိုင်အတိုးဖြင့် သိမ်းဆည်းပါ။
  3. ဖန်တီးထားသော HTML ဖိုင်ကို ဝဘ်ဘရောက်ဆာတွင် ဖွင့်၍ စာပိုဒ်များကို ဖတ်ရှုပါ။
