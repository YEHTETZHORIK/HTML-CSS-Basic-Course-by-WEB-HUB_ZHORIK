Tables

HTML တွင် အချက်အလက်များကို စာရင်းဇယားများအဖြစ် ဖော်ပြရန် <table> tag ကို အသုံးပြုသည်။ <table> tag ၏ အတွင်းတွင် <tr> (table row), <th> (table header), <td> (table data) တို့ကို အသုံးပြုပြီး ဇယားကို ဖော်ပြနိုင်သည်။

HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Tables Example</title>
</head>
<body>
    <table>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>City</th>
        </tr>
        <tr>
            <td>John</td>
            <td>25</td>
            <td>New York</td>
        </tr>
        <tr>
            <td>Jane</td>
            <td>30</td>
            <td>Los Angeles</td>
        </tr>
        <tr>
            <td>Mike</td>
            <td>35</td>
            <td>Chicago</td>
        </tr>
    </table>
</body>
</html>

Result

Name Age City
John 25 New York
Jane 30 Los Angeles
Mike 35 Chicago

ရှင်းပြချက်

HTML တွင် <table> tag ကို အသုံးပြုပြီး စာရင်းဇယားများကို ဖော်ပြနိုင်သည်။ <tr> tag ကို အသုံးပြုပြီး ဇယားတန်းတစ်ခုစီကို ဖော်ပြပါ။ <th> tag ကို ဇယားခေါင်းစဉ်အချက်များအတွက် အသုံးပြုပြီး <td> tag ကို ဇယားတွင်းအချက်အလက်များအတွက် အသုံးပြုပါ။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. HTML ဖိုင် အသစ်ဖန်တီးပြီး <table> tag ကို အသုံးပြုပြီး သင်၏ မိတ်ဆွေများ၏ အချက်အလက်များကို စာရင်းဇယားပြုစုပါ။
  2. <tr>, <th>, <td> tag များကို အသုံးပြုပြီး ဇယား၏ တန်းများနှင့် အချက်အလက်များကို သတ်မှတ်ပါ။
  3. ဇယားအတွက် စိတ်ကြိုက်ဒီဇိုင်းအရောင်များကို ထည့်သွင်းပြင်ဆင်ပါ။
